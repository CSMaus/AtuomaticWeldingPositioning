1. Python 3.9._이 설치되어 있는지 확인하세요. "제어판" -> "프로그램 및 기능"에서 확인할 수 있습니다. 
1.1 만약 Python이 설치되어 있지 않다면, 공식 웹사이트(저는 3.9.7 버전 사용: https://www.python.org/downloads/release/python-397/)에서 다운로드 후 설치할 수 있습니다.

참고: 다른 버전의 Python이 설치되어 있다면, Python을 재설치하기 전에 requirements를 설치하고 프로그램을 실행해 볼 수 있습니다.

2. 필요한 라이브러리를 설치하려면, 이 폴더에서 명령 프롬프트를 열고 다음 명령어를 실행하세요:
pip install -r requirements.txt

3. 프로그램을 실행하려면, 다음 명령어를 입력하세요:
python main_gui.py
프로그램 실행 전에 카메라가 연결되어 있는지 확인하세요.

4. 프로그램을 종료하려면, 창을 닫거나 명령 프롬프트에서 Ctrl + C를 누르세요.
